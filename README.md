# clober-market-maker

To install dependencies:

```bash
bun install
```

Setting private key in `.env`
```text
PRIVATE_KEY=0x...
RPC_URL=
```

To run:

```bash
CHAIN_ID=421614 bun run index.ts
```

This project was created using `bun init` in bun v1.0.20. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
